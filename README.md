Skin changer for taming.io
This script allows you to change skins easier and comfortable, press K to open menu.

## How to use:
- Donwload newest version from Releases.
- Put it in tampermonkey.
- Join in game.

The game dev's active patching this skin changer, use it on your risk. Your account might be banned.
